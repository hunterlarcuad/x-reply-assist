Privacy Policy - AssistX

Effective Date: April 29, 2026
Last Updated: April 29, 2026

1. Overview

AssistX ("we", "our", or "the Extension") is a Chrome extension that provides AI-powered reply suggestions for Twitter/X. We are committed to protecting your privacy and being transparent about how we handle your data.

2. Information We Collect

Data TypePurposeStorageShared With
API Keys To authenticate with your chosen AI service provider Locally via chrome.storage.local Sent only to your configured AI API endpoint
Settings & Preferences To remember your model selection, language, UI preferences, custom prompts Locally via chrome.storage.local Not shared
Tweet Content (temporary) To generate contextually relevant AI replies In memory only during processing Sent only to your configured AI API endpoint as part of the prompt

3. Information We Do NOT Collect

  • No user accounts or personal identifiers
  • No usage analytics or tracking
  • No cookies or third-party trackers
  • No data sent to our servers
  • No access to your browsing history beyond active Twitter/X tabs

4. How Data Is Processed

  1. Tweet Extraction: When you trigger a reply generation, the extension reads the currently visible tweet content from the active Twitter/X tab.
  2. API Call: The tweet content (as a prompt) and your API key are sent directly to the AI service endpoint you have configured (e.g., ZhipuAI, OpenAI, Anthropic).
  3. Response: The AI-generated reply is displayed in the extension's UI. No response data is stored persistently.

5. Third-Party Services

This extension sends data to third-party AI service providers that you explicitly configure:

  • ZhipuAI (bigmodel.cn) — When using GLM models
  • OpenAI — When using GPT models
  • Anthropic — When using Claude models
  • Other providers — Any custom API endpoint you configure

Each provider's own privacy policy applies to data sent to their services. We recommend reviewing their policies before use.

6. Data Retention

All data stored by this extension (chrome.storage.local) remains on your local device until:

  • You actively delete it through the extension's settings
  • You uninstall the extension (Chrome will remove all stored data)
  • You clear your browser data

7. Your Rights

  • Export: You can view and copy all stored settings from the extension's settings panel.
  • Delete: Uninstalling the extension immediately removes all stored data.
  • Opt-out: Simply do not enter an API key or uninstall the extension.

8. Security

  • API keys are stored using Chrome's secure storage.local API, which isolates data per extension.
  • All API communication uses HTTPS/TLS encryption.
  • The extension does not include any remote code loading or update mechanisms outside of the Chrome Web Store.

9. Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect personal information from children.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated revision date.
